# templates/

HTML templates used as the starting point for new pages.

Planned templates:
- `country-page.html` — template for a country index page.
- `report-initial.html` — template for an Initial Market Analysis report page.
- `report-update.html` — template for a Periodic Market Update report page.

Templates are added here when a new page type is needed for the first time.
Do not publish template files directly — copy and populate them into the correct `countries/` subfolder.
